# 文本分类实验结果整理

> 本文件根据你提供的多张终端截图整理。  
> 任务背景：TNEWS / 中文新闻 15 分类。  
> 主要对比对象：BERT fine-tune、Qwen2-0.5B zero-shot、Qwen2-0.5B SFT LoRA。

---

## 实验结果总览

| 方法                 |             训练/评估样本 | 训练方式               | Checkpoint / 模型     | Accuracy | Macro F1 | Weighted F1 |         无法解析 | 总耗时 | 平均耗时 |
| -------------------- | ------------------------: | ---------------------- | --------------------- | -------: | -------: | ----------: | ---------------: | -----: | -------: |
| BERT fine-tune       | train 53,360 / val 10,000 | 全量监督微调，3 epochs | `best_cls.pt`         |   0.5719 |   0.5581 |        0.57 |                - |      - |        - |
| BERT fine-tune       | train 53,360 / val 10,000 | 全量监督微调，3 epochs | `best_mean.pt`        |   0.5623 |   0.5408 |        0.56 |                - |      - |        - |
| Qwen2-0.5B zero-shot |                       200 | 不训练，直接提示词分类 | `Qwen2-0.5B-Instruct` |   0.3600 |        - |           - | 58 / 200 = 29.0% |  24.2s | 0.12s/条 |
| Qwen2-0.5B SFT LoRA  |                       200 | LoRA SFT，小样本微调   | `Qwen2-0.5B-Instruct` |   0.5800 |        - |           - |   2 / 200 = 1.0% |  33.3s | 0.17s/条 |

### 直接结论

1. **BERT fine-tune 的验证集准确率约 0.56～0.57。**
2. **Qwen2-0.5B zero-shot 在 200 条样本上准确率只有 0.36，且无法解析率高达 29%。**
3. **Qwen2-0.5B 经过 200 条样本 LoRA SFT 后，准确率提升到 0.58，无法解析率降到 1%。**
4. 在这组实验里，**小样本 SFT 后的 Qwen2-0.5B 在 200 条随机验证样本上已经接近甚至略高于当前 BERT fine-tune 的 val accuracy**，但注意二者评估规模不同：BERT 是 10,000 条验证集，Qwen SFT 这里是 200 条样本。
