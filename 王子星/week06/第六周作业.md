# 文本分类：多模型同文本标注对比实验报告

## 1. 实验概述

本报告针对 **新闻标题分类数据集**（15 类），对同一批文本分别使用三种方法进行标注，直观对比各方法的分类行为与性能差异。

| 方法 | 模型 | 向量/生成策略 | 可训练参数 |
|------|------|-------------|-----------|
| BERT Fine-tune | bert-base-chinese | **CLS 池化** | 102.3M（全参微调） |
| LLM Zero-shot | Qwen2.5-0.5B-Instruct | 生成式（无训练） | 0（仅推理） |
| LLM LoRA SFT | Qwen2.5-0.5B-Instruct + LoRA | 生成式（LoRA 微调） | ~1.08M（0.22%） |

## 4. LLM Zero-shot 结果（Qwen2.5-0.5B-Instruct）

### 4.1 推理配置

```

### 4.2 Prompt 模板

```
你是一个新闻分类助手。请将以下新闻标题分类到以下15个类别之一：

## 8. 结论

1. **BERT Fine-tune 仍是最稳定方案**：全量验证集准确率 55.98%，无"无法解析"问题，推理延迟毫秒级，适合高并发实时场景。
2. **LoRA 以 0.22% 参数量逼近 BERT 水平**：仅用 5K 训练数据（BERT 的 9.4%）即达到 54.00%，在同文本 15 条样本上准确率高达 93.3%，参数效率极高。
3. **Zero-shot 存在严重的标签对齐问题**：500 条样本评估显示 43.6% 输出无法解析，准确率仅 28.8%，0.5B 小模型不适合直接用于生产分类任务。
4. **同文本对比揭示核心差异**：LoRA SFT 在少数类（证券）和语义歧义类（文化 vs 旅游）上均优于 BERT 和 Zero-shot，说明指令微调能有效弥合预训练分布与任务标签体系之间的差距。
5. **改进方向**：将 LoRA SFT 训练数据从 5K 扩展至全量 53K 有望超越 BERT；引入加权 CrossEntropyLoss 可进一步提升 BERT 对少数类的 Recall（加权 Loss 在 3 epoch 下 Macro F1 达 55.34%，高于普通 Loss 的 54.30%）。
