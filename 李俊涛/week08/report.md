实验核心结论与主要指标
通过在 AFQMC 数据集上的对比实验，我们完整量化了交互式编码器与双编码器在精度与效率上的权衡。实验最终取得了优异的微调效果：
Cross-Encoder 在第 3 轮迭代达到最优性能，验证集准确率达 88.20%，F_1 分数达 88.20%。
Bi-Encoder  在第 3 轮迭代达到最优性能，验证集准确率达 86.31%，F_1 分数达 86.30%。
速度与精度的权衡 
Cross-Encoder： 准确率领先 Bi-Encoder 约 1.89个百分点。由于其在浅层就实现了 Query 和 Document 的信息交融，特征捕获更细腻，但无法预先计算向量，在线计算复杂度随候选集规模呈线性爆炸增长。
Bi-Encoder： 牺牲了少许精度，但由于句向量可以离线预计算并存入向量数据库，在线推理时仅需计算一次 Query 向量和线性的余弦相似度，适合大规模检索。
负采样策略与 Pooling 验证
负样本挖掘： 针对 AFQMC 数据集非平衡性（正负比 31:69），通过离线负采样构建的 10K 三元组为 TripletLoss 奠定了基础。实验验证了在线难负样本挖掘能显著提升模型对易混淆问句对的判别边界，表现优于随机负采样。
Pooling 策略： 实验验证了 Sentence-BERT 的经典结论：Mean Pooling输出的句向量，在下游余弦相似度度量中，表现显著优于仅取 [CLS] 位置的表征。
层数剪裁： 通过限制 BERT 层数（4层 到 12层），量化了剪枝/蒸馏带来的加速比，发现在特定的浅层配置下，能够以极小的精度损失换取数倍的推理提速
